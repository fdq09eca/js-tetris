# TetrisJS

Simple Tetris in JS

## How to play

### Control

- :arrow_down: :arrow_left: :arrow_right:
- Hard Drop: <kbd>space</kbd> 
- Hold Piece: <kbd>ctrl</kbd>
- Pause: <kbd>esc</kbd>
- Restart: <kbd>r</kbd>
