# TetrisJS

Simple Tetris in JS

## How to play

### Control

> [!NOTE]  
> :arrow_down: :arrow_left: :arrow_right: 
> Hard Drop: <kbd>space</kbd>

> [!TIP]
> Pause: <kbd>esc</kbd>
> Restart: <kbd>r</kbd>
